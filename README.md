# teste-rv

## Rodar o projeto local

```
git clone https://github.com/airton/teste-rv.git
```
### Rodar comandos

```
npm install
```

```
bower install
```

### Rodar no Browser

```
gulp sync
```
> Alterar __Gulpfile.js__ na linha 159 para a url do seu projeto local `proxy: "local.teste-rv"`

### Acessar url

[http://localhost:3000/](http://localhost:3000/)
